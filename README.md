# CodeAlpha Web Scraping Project

## Objective
Extract data from a website using Python web scraping techniques and store/process the collected information.

## Technologies Used
- Python
- Requests
- BeautifulSoup4
- Pandas

## Features
- Fetches webpage content
- Extracts required data
- Cleans and organizes data
- Displays results in a structured format

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python scraper.py
